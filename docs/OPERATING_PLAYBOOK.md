# TalentScope 岗位补位运营手册

## 目标

- 优化 HR 外包人才库简历档案质量
- 持续提升岗位与人才库匹配度
- 在岗位流动发生时快速补位，降低空岗风险
- 支持新入职简历无缝接入同一流程

## 日常流程

1. 人才入库
- HR 在“简历库管理”页批量导入新增简历
- 指定部门并检查解析结果
- 必要时补充备注，建立结构化档案

2. 岗位补位
- 在“JD 匹配评分”页输入岗位需求
- 选择人才库候选人并设置补位目标人数
- 运行评分，查看 A/B/C 后备池和连续性风险

3. 面试与核验
- 使用系统给出的评分证据、风险提示、建议追问
- 完成技术面和业务面复核

4. 反馈沉淀
- 在“简历库管理”页写回通过/存疑/淘汰等反馈
- 每周导出反馈评测 CSV 或离线评测报告

## 周期复盘指标

- 反馈覆盖率：目标 >= 70%
- 正向反馈率：观察岗位模板有效性
- 负向反馈率：定位误判来源并回调门槛
- A 层后备池数量：至少覆盖关键岗位补位目标

## 角色分工

- HR：入库、筛选、反馈更新
- 用人经理：岗位要求确认、面试核验
- 技术支持：模型配置、词典维护、系统巡检

## 变更管理建议

- 所有配置变更记录到变更日志
- 评分门槛调整采用小步迭代
- 每次改动后用样本岗位做回归验证
